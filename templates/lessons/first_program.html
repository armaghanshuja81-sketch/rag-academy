Your First Python Program

Open the Python Playground and type this:

# This is a comment - Python ignores it
print("Hello, World!")
print(2 + 2)
print("The answer is", 42)

What Just Happened?

  • print() is a function that displays text
  • Text inside quotes is a string
  • Numbers without quotes are integers
  • Python runs code line by line, top to bottom

Try This:

  1. Change the message to say your name
  2. Add a line that prints your age
  3. Make Python multiply 8 by 7: print(8 * 7)
