What is Flask?

Flask is a lightweight Python web framework. It turns your Python code into a web server.

The Minimal Flask App

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, World!"

if __name__ == "__main__":
    app.run(debug=True)
# Run: python app.py
# Open: http://localhost:5000

How It Works

  1. You write Python functions
  2. You attach them to URLs with @app.route()
  3. When someone visits that URL, Flask runs your function
  4. Your function returns HTML/text to the browser

Routes Map to URLs

@app.route("/")         # http://localhost:5000/
@app.route("/about")    # http://localhost:5000/about
@app.route("/user/<name>")  # Dynamic: /user/Armaghan
This entire app IS a Flask app! Every page you visit follows this pattern: URL → route → function → HTML template → response.
