Git & GitHub Basics

Git tracks changes to your code. Think of it like 'Save As' on steroids. Every commit is a saved version.

Essential Commands

git init                    # Start tracking a folder
git add .                   # Stage all files
git commit -m "First commit"
git remote add origin [URL]
git push
