CRUD Operations

Every database operation is one of these four:

CREATE (INSERT) - Add Data

INSERT INTO users (name, email, age)
VALUES ('Armaghan', 'armaghan@email.com', 20);

READ (SELECT) - Retrieve Data

SELECT * FROM users;                         -- All rows
SELECT name, email FROM users;               -- Specific columns
SELECT * FROM users WHERE age > 18;          -- Filtered
SELECT * FROM users ORDER BY name DESC;      -- Sorted
SELECT COUNT(*) FROM users;                  -- Count rows

UPDATE - Modify Data

UPDATE users SET age = 21 WHERE id = 1;

DELETE - Remove Data

DELETE FROM users WHERE id = 1;
WARNING: Always use WHERE with UPDATE and DELETE!
DELETE FROM users; - Deletes EVERY row!

From Python

import sqlite3
conn = sqlite3.connect("myapp.db")
cursor = conn.cursor()

# ALWAYS use ? placeholders (prevents SQL injection!)
cursor.execute("SELECT * FROM users WHERE email = ?", ("a@b.com",))
rows = cursor.fetchall()
conn.close()
