Embeddings & Cosine Similarity

Embeddings convert text to number vectors. Similar texts have similar vectors.

import numpy as np
def cosine_sim(a, b):
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

vec1 = [0.1, 0.3, 0.5]  # Embedding of "What is RAG?"
vec2 = [0.4, 0.1, 0.2]  # Embedding of document chunk
score = cosine_sim(vec1, vec2)  # 0.0 to 1.0
