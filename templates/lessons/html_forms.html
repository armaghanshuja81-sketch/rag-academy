HTML Forms & Data Flow

Forms are how data gets from your browser to your server. This is the most important HTML concept for backend developers.

The Critical Attributes

<form action="/submit" method="POST">
    <!-- action: WHERE to send (Flask route URL)
         method: HOW to send (GET or POST) -->

    <label for="username">Name:</label>
    <input type="text" id="username" name="username">
    <!-- ^^ name="username" is THE CONNECTION to Python!
         Flask reads: request.form["username"] -->

    <button type="submit">Send</button>
</form>

Input Types

<input type="text">     <!-- Single line text -->
<input type="email">    <!-- Email validation -->
<input type="password"> <!-- Hidden text -->
<input type="number">   <!-- Numbers only -->
<input type="file">     <!-- File upload -->
<textarea></textarea>   <!-- Multi-line text -->
<select>                <!-- Dropdown -->
    <option value="1">One</option>
</select>

Flask Side

@app.route("/submit", methods=["POST"])
def submit():
    username = request.form["username"]
    uploaded_file = request.files.get("file")
    return f"Received: {username}"
KEY INSIGHT: The name="" in HTML and request.form[""] in Flask MUST match exactly. This is the data bridge between frontend and backend.
