🐍 Flask: The Python Web Framework

Flask turns Python code into a web server. It handles HTTP requests and returns HTML responses.

What is Flask?

Flask is a micro web framework for Python. It lets you write web apps using Python.

  • Route — A URL path like / or /about
  • View function — A Python function that runs when someone visits a route
  • Template — An HTML file with Jinja2 placeholders ({% raw %}{{ var }}{% endraw %})

The Minimal Flask App

Python 
from flask import Flask
app = Flask(__name__)

@app.route("/")          # When someone visits http://localhost:5000/
def home():              # This function runs
    return "Hello!"      # This text is sent to the browser

if __name__ == "__main__":
    app.run(debug=True)  # Start the server

Run with: python app.py → Visit http://localhost:5000

How Flask Receives Form Data

This is the critical connection — how HTML forms talk to Python:

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/ask", methods=["GET", "POST"])
def ask():
    if request.method == "POST":
        # Get data sent from HTML form
        user_question = request.form["question"]
        # ↑ This "question" must match name="question" in HTML!

        # Do something with the data
        answer = f"You asked: {user_question}"

        # Send result back to browser
        return render_template("result.html",
                             question=user_question,
                             answer=answer)

    # GET request: just show the form
    return render_template("ask.html")
⚠️ Common Mistake: If your HTML form has method="POST" but your Flask route only has @app.route("/ask") (no methods), you'll get a 405 Method Not Allowed error. Always add methods=["GET", "POST"] when handling forms.

Template Variables (Jinja2)

Flask uses Jinja2 to embed Python variables in HTML:

{# templates/result.html #}
<!DOCTYPE html>
<html>
<head><title>Result</title></head>
<body>
    <h1>Your Question: {% raw %}{{ question }}{% endraw %}</h1>
    <p>Answer: {% raw %}{{ answer }}{% endraw %}</p>
</body>
</html>

The {% raw %}{{ }}{% endraw %} syntax prints Python variables as text in your HTML.

  1. You click a link or submit a form
  2. Browser sends HTTP request to Flask

  4. Flask renders an HTML template with data
  5. You see the rendered page
