Variables & Data Types

A variable is a container that stores data - like a labeled box.

# Creating variables
name = "Armaghan"        # String (text)
age = 20                 # Integer (whole number)
height = 5.9             # Float (decimal)
is_student = True        # Boolean (True/False)

# Using them
print(name)              # Prints: Armaghan
print(age + 5)           # Prints: 25
print(type(age))         # Prints: <class 'int'>

# Variables can change!
age = 21
print(age)               # Now 21

Naming Rules

  • Use lowercase with underscores: my_variable
  • Cannot start with a number: 1name is invalid
  • Cannot use Python keywords: if, for, while
