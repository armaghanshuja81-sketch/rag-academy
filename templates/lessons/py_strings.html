Strings & String Methods

A string is a sequence of characters.

name = "Armaghan"
full = name + " Khan"  # Concatenation
print(f"Hello, {name}")  # f-strings
print(name.upper())       # ARMAGHAN
print(len(name))          # 8
print(name[0:4])          # Arma (slicing)

Strings are everywhere in RAG - you'll constantly clean text, split documents, and search for keywords.
