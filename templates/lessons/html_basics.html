📝 HTML Basics: The Language of the Web

HTML is the foundation of every webpage. It structures content using "tags".

What is HTML?

HTML = HyperText Markup Language.

  • It's NOT a programming language — it's a markup language
  • It uses tags like <h1> and <p> to structure content
  • The browser reads HTML and renders it as a visual page

Every HTML Page Has This Structure

HTML 
<!DOCTYPE html>      <!-- Tells browser: "I'm HTML5" -->
<html>                <!-- Root element -->
<head>                <!-- Metadata (not visible) -->
    <title>My Page</title>  <!-- Browser tab title -->
</head>
<body>                <!-- Visible content -->
    <h1>Hello World!</h1>   <!-- Heading level 1 -->
    <p>This is a paragraph.</p>
</body>
</html>

The Form Tag — The Most Important One

Forms are how data gets FROM the browser TO the server:

<form action="/submit" method="POST">
    <!-- action: WHERE to send data -->
    <!-- method: HOW to send (GET or POST) -->

    <label for="name">Name:</label>
    <input type="text" id="name" name="username">
    <!--  ↑ name="username" is CRITICAL — that's
    │      how Flask identifies this field!  -->

    <button type="submit">Send</button>
</form>
KEY INSIGHT: The name="" attribute is the link between HTML and Python. Flask reads request.form["username"] — that "username" must match the name="username" in your HTML.
