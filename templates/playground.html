📖 How the Playground Works

The playground demonstrates a core web concept: client-side rendering.

  1. Left panel: You write HTML in a <textarea>
  2. Your browser detects every keystroke
  3. JavaScript captures the HTML and creates a Blob (a virtual file)
  4. An <iframe> loads that Blob as if it's a real webpage
  5. Right panel: Shows your rendered page — instantly!
Full-Stack Insight: The right panel is essentially what a browser does with any HTML it receives from a server. The only difference is this HTML came from YOUR keyboard, not the internet.
