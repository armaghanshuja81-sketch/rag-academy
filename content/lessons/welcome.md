## Welcome to RAG Academy!

This complete learning platform takes you from **absolute beginner** to **professional RAG developer**.

### What You Will Learn

1. **Python** — The language of AI and data science
2. **HTML/CSS** — How web pages work and send data
3. **Flask** — Python web framework for building apps
4. **Databases & SQL** — Storing and querying data
5. **AI & LLMs** — How ChatGPT and similar models work
6. **RAG from Scratch** — Build it yourself, understand every piece
7. **Vector Databases** — ChromaDB, FAISS, and more
8. **LangChain** — The industry standard RAG framework
9. **Advanced RAG** — Production-ready techniques
10. **Career** — Freelance, certifications, portfolio

### How to Use This App

- Work through lessons **in order** — each builds on the last
- **Type every code example** yourself in the Python Playground
- **Watch the videos** linked in the Resources page
- **Mark lessons complete** to track your progress
